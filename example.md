这是一个包含表格和 LaTeX 公式的 Markdown 示例。

## 一些常见的数学公式

1. **二次方程公式**：
   \[
   ax^2 + bx + c = 0
   \]

2. **勾股定理**：
   \[
   a^2 + b^2 = c^2
   \]

3. **欧拉公式**：
   \[
   e^{i\pi} + 1 = 0
   \]

4. **积分公式**：
   \[
   \int_a^b x^2 \, dx = \frac{b^3}{3} - \frac{a^3}{3}
   \]
